### 🔹 一、数组 & 双指针（10题）
1. **两数之和（Two Sum）** → 哈希表记录值-索引
2. **三数之和（3Sum）** → 排序 + 双指针去重
3. **盛最多水的容器** → 对撞双指针（贪心移动短边）
4. **接雨水** → 双指针 or 单调栈（短板效应）
5. **滑动窗口最大值** → 单调队列（维护窗口内递减序列）
6. **长度最小的子数组（和 ≥ target）** → 滑动窗口（同向双指针）
7. **合并两个有序数组** → 从后往前双指针（避免覆盖）
8. **移动零** → 快慢指针（快指针找非零，慢指针放位置）
9. **删除有序数组中的重复项** → 快慢指针（慢指针记录不重复位置）
10. **轮转数组（右移k位）** → 三次翻转 or 环状替换

---

### 🔹 二、链表（8题）
11. **反转链表** → 迭代（pre/cur/next三指针）or 递归
12. **合并两个有序链表** → 虚拟头节点 + 双指针
13. **环形链表（判断是否有环）** → 快慢指针（Floyd判圈）
14. **环形链表 II（找入环点）** → 快慢指针 + 数学推导
15. **相交链表** → 双指针走完各自再走对方（等长相遇）
16. **删除链表的倒数第N个节点** → 快慢指针（快先走N步）
17. **回文链表** → 快慢指针找中点 + 反转后半段比较
18. **复制带随机指针的链表** → 哈希映射 or 原地拼接拆分

---

### 🔹 三、二叉树（10题）
19. **二叉树的前/中/后序遍历（迭代）** → 栈模拟递归（中序：左到底；后序：加标记或逆前序）
20. **层序遍历（BFS）** → 队列 + 按层处理（记录size）
21. **二叉树的最大深度** → DFS递归 or BFS层数
22. **验证二叉搜索树（BST）** → 中序遍历递增 or 递归带范围
23. **对称二叉树** → 双指针递归（左子树 vs 右子树镜像）
24. **二叉树的最近公共祖先（LCA）** → 后序遍历（自底向上找）
25. **路径总和（是否存在根到叶路径和=target）** → DFS回溯
26. **从前序与中序遍历序列构造二叉树** → 分治（前序定根，中序分左右）
27. **二叉树展开为链表（前序）** → 递归（右子树挂到左子树最右）
28. **二叉搜索树中第K小的元素** → 中序遍历（计数到K）

---

### 🔹 四、回溯 & DFS（6题）
29. **全排列** → 回溯（用used数组标记）
30. **组合总和（可重复选）** → 回溯（start控制不重复组合）
31. **子集** → 回溯（每个元素选/不选）
32. **括号生成** → 回溯（控制左右括号数量）
33. **N皇后** → 回溯（行递进 + 列/对角线冲突检查）
34. **单词搜索（二维网格）** → DFS + 回溯（标记访问）

---

### 🔹 五、动态规划（DP）（8题）
35. **爬楼梯（斐波那契）** → 一维DP（dp[i] = dp[i-1] + dp[i-2]）
36. **打家劫舍** → 一维DP（偷/不偷当前，状态转移）
37. **最大子数组和（Kadane）** → 一维DP（dp[i] = max(nums[i], dp[i-1]+nums[i])）
38. **买卖股票的最佳时机（一次交易）** → 记录历史最低价
39. **最长递增子序列（LIS）** → 一维DP（dp[i] = max(dp[j]+1) for j<i and nums[j]<nums[i]）
40. **零钱兑换（最少硬币数）** → 完全背包（dp[amount] = min(dp[amount-coin]+1)）
41. **编辑距离** → 二维DP（插入/删除/替换三种操作）
42. **不同路径（机器人走格子）** → 二维DP（只能右/下，dp[i][j]=dp[i-1][j]+dp[i][j-1]）

### 🔹 六、其他高频（8题）
43. **LRU缓存** → 哈希表 + 双向链表（O(1) get/put）
44. **实现 Trie（前缀树）** → 多叉树结构（每个节点存children和isEnd）
45. **岛屿数量（DFS/BFS）** → 网格遍历 + 沉岛（标记visited）
46. **课程表（拓扑排序）** → DFS（检测环）or BFS（入度为0入队）
47. **Pow(x, n)（快速幂）** → 递归分治（n偶：x²^(n/2)；n奇：x * x²^((n-1)/2)）
48. **寻找峰值** → 二分查找（比较mid和mid+1决定方向）
49. **多数元素（>n/2）** → 摩尔投票法（抵消思想）
50. **有效的括号** → 栈（匹配弹出，最后空栈）
